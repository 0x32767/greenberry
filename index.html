INTRODUCTION

greenBerry
is a one-line statement programming language with extention .gb
Simple and easy.

On the technical side, it is based on python and supports infinite look-ahead !

SYNTAX


variable declaration

var x = 1

printing

one word
print a
>>> a

eval
print eval (2+3-4)
>>> 1

string
print string a cat walks along the path
>>> a cat walks along the path

variable referencing
print @x
>>> 1

plotting

plot 1,2,3,4,5 x-time 23,34,56,78,45 y-letters
>>>

minified for loop

for 3 times : print greenBerry
>>> greenBerry
>>> greenBerry
>>> greenBerry

equality

>>> leaf
