## Run files

### 🔧 REPL 

Just run `python gb_repl.py`

### 📁 .gb file

Write in *main.gb* and then save and run `python gb_main.py`

### 🖥️ IDE

Run `python gb_ide.py` write and run or load a *.gb* file !
